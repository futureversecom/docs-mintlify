<CodeGroup>
  ```npm npm

  npm install wagmi viem@2.x @tanstack/react-query --save
  ```

  ```yarn yarn
  yarn add @futureverse/auth-react @futureverse/auth-ui
  yarn add wagmi viem@2.x @tanstack/react-query
  ```

  ```pnpm pnpm
  pnpm add @futureverse/auth-react @futureverse/auth-ui
  pnpm add wagmi viem@2.x @tanstack/react-query
  ```

  ```bun Bun
  bun add @futureverse/auth-react @futureverse/auth-ui
  bun add wagmi viem@2.x @tanstack/react-query
  ```
</CodeGroup>

## Prerequisites

Pass React requires an installation of a few peer dependencies:

* [Auth UI](https://www.npmjs.com/package/@futureverse/auth-ui) - a React theme UI provider for Pass React.

* [Wagmi](https://www.npmjs.com/package/wagmi) - React Hooks for Web3

* [Viem](https://www.npmjs.com/package/viem) - TypeScript Interface for Web3

* [TanStack](https://www.npmjs.com/package/@tanstack/react-query) - Hooks for fetching, caching and updating asynchronous data in React.
